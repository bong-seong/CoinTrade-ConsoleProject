# coinProject
2개월차 console coinProject


1. 주제 : 코인 매입 (console)


2. 주제 선정 이유
* CRUD 를 학습하고 이해할 수 있다.
* 체계적인 MVC 모델을 설계하고 이를 활용 할 수 있다. 
* 스레드 개념을 이해하고 멀티스레드를 활용 할 수 있다.


3. 팀별 일정

* 2/24 : 주제 선정 및 DB 설계

4. 각 조원별 역할분담 / 일정

- 최성아 : 계좌 잔고 확인 , 입금 , 출금

- 권태형 : 코인 출력 페이지 , 코인 가격 변동 

- 김성봉 : 코인 매수 , 코인 매도 , 손익 계산 , 코인가격 변동

- 김태호 : 회원가입 , 로그인 , 계좌 생성 , 코인 등록 , 회원 탈퇴
